## 2. Steps To Follow:

### Conda
```
git clone https://github.com/dragonblood/Cataract.git
cd Cataract
conda env create --file cataract.yml
conda activate Cataract
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
### ~~~~~~~ OR ~~~~~~~

### PIP
```
git clone https://github.com/dragonblood/Cataract.git
cd Cataract
pipenv shell
pipenv install
python manage.py makemigrations
python manage.py migrate
python manage.py runserver
```
